## moogoose

    学习数据库操作

### 主要涉及内容

    - mongodb
    - mongoose
    - koa

### 1.项目初始结构

    1.安装本地数据库mongodb
    2.安装mongodb的可视化工具 compass
    3.使用koa搭建一个简单的应答服务器
    4.scheme新建数据库(文档约束模型)
    5.model建立数据库集合
    6.mogoose连接数据库
    7.实例化一个文档对象
    8.调用实例上的方法save将数据保存到数据库中
    9.调用model上的find的方法查询数据库
